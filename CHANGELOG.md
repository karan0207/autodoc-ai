# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AutoDoc AI
- Web crawling with Playwright
- GitHub repository integration
- HuggingFace embeddings support
- Weaviate vector database integration
- OpenAI GPT-4 documentation generation
- Next.js frontend with Shadcn UI
- Multiple documentation types:
  - API Reference
  - Product Description
  - Changelog Summary
  - SEO Landing Page
- Export to Markdown functionality
- Clipboard copy support

### Changed
- Migrated from Jina AI to HuggingFace for embeddings
- Optimized chunking strategy for better context

### Fixed
- Async/await handling in Windows environment
- Crawler URL validation logic
